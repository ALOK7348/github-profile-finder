# GitHub Profile Finder

A GitHub Profile Finder built using HTML, CSS, JavaScript and GitHub REST API.

## Features

- Search GitHub users
- Display profile image
- Followers count
- Following count
- Public repositories
- Dynamic profile rendering
- Error handling for invalid usernames
- Attractive responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub REST API

## Project Flow

User enters username
↓
Fetch API request
↓
GitHub API
↓
JSON response
↓
UI updates dynamically
